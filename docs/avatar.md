# 새로운 아바타 만들기
## 아바타 아틀라스 만들기
새로운 아바타를 추가하려면 아바타의 **아틀라스 (스프라이트 시트) 파일**이 필요해요.

아래 이미지를 참고해서 아틀라스 파일을 만들어주세요.
<br>아틀라스의 모든 이미지(프레임)는 같은 가로x세로 크기를 가져야 합니다.

<img width="490" height="672" alt="image" src="https://github.com/user-attachments/assets/a6587050-17f3-43e5-b7fb-429fb8016742" />


| 행 번호 | 이름 | 사용되는 액션 |
|--------|-----|------------|
|1|``idle``| 가만히 서 있는 상태 |
|2|``walk`` | 좌우 이동 중 |
|3|``jump``| ``!점프``, ``!jump`` 명령어로 점프한 경우 |
|4|``fall``| 높은 곳에서 떨어진 경우, 마우스 드래그로 던져질 경우 | 
|5|``grab``| 캐릭터가 마우스에 잡힌 경우 (클릭된 경우) |
|6|``sit``| ``!sit``, ``!앉아`` 명령어로 앉는 경우 / 앉아있는 상태 |
|7|``stand``| ``!stand``, ``!일어나`` 명령어로 일어난 경우 |
> [!NOTE]
> - 아틀라스의 높이가 7행을 넘는 경우, 초과하는 행들은 ``row_(행숫자)`` 라는 애니메이션으로 저장됩니다.
>   <br>해당 애니메이션들은 직접 소스를 수정하거나 따로 업데이트가 있지 않은 한, 오버레이에서 사용되지 않습니다.

<br>

## 아바타 파일 만들기
캔냥닷치 오버레이를 실행하고, ``아바타 관리`` 탭에서 ``새로 만들기`` 를 클릭합니다.

<img width="680" height="542" alt="image" src="https://github.com/user-attachments/assets/0ed911fa-c67c-4cd2-b3b8-66fbc8bc1883" />

1. `...` 버튼을 눌러 아바타 아틀라스 파일을 불러옵니다.
<br> 아바타의 가로x세로 해상도를 입력하고, 프레임당 재생 시간(재생 속도)을 입력합니다.

2. 입력값이 맞는지 확인하고 하단의 ``만들기`` 버튼을 누르면 창의 오른쪽에 아바타의 미리보기가 표시됩니다.
<br> 미리보기 영역 위의 드롭다운 메뉴를 활용하여 각각의 애니메이션을 확인할 수 있습니다.

3. 아바타의 메타데이터 (아바타 이름, 제작자, 아바타에 대한 간단한 설명)를 아래 입력란에 입력해주세요.
<br>아바타 이름 / 제작자는 공백을 제외한 한글/영숫자로 최대 10자까지 입력 가능합니다.

4. 아래의 ``저장하기`` 버튼을 눌러 아바타 파일(.res)을 원하는 경로에 저장해주세요.

> [!NOTE]
> - v1.1 기준 프레임당 최대 100x100px를 지원합니다.
> - 재생 속도는 1-10의 값을 가지며, 10에 가까워질수록 재생 속도가 빨라집니다. (추천값: 2~3)
> - 메타데이터를 입력하지 않을 경우 ``Unknown Avatar`` 로 자동 저장됩니다.

# 아바타 관리하기
<img width="311" height="715" alt="image" src="https://github.com/user-attachments/assets/054da47c-14c2-490e-8918-c79c1e9013e9" />

## 아바타 파일 불러오기
캔냥닷치 오버레이를 실행하고, ``아바타`` 탭에서 ``불러오기`` 버튼을 클릭합니다.
<br>올바른 아바타 리소스 파일 (.res 파일) 을 선택해주세요.

<br>

## 아바타 기본값 설정하기
> 방송에 처음 접속하는 시청자들이 어떤 아바타를 기본으로 사용할지 설정할 수 있습니다.

``아바타`` 탭의 ``등록된 아바타 목록``에서 기본값으로 설정할 아바타를 클릭합니다.

<br>아바타 관리 화면 아래의 ``기본 아바타로 설정`` 체크박스를 활성화합니다. 기본값 설정된 아바타는 이름 뒤에 ``*`` 표시가 붙습니다.
<br>다른 아바타를 기본값으로 설정하면 자동으로 기존 아바타의 기본값 옵션은 해제됩니다.

> [!NOTE]
> - 프록시에 접속하기 전, 최소 1개의 아바타는 기본 아바타로 등록되어 있어야 합니다.
> - 기본 아바타로 등록된 아바타는 삭제가 불가능합니다. 아바타를 삭제하기 전 기본값 설정을 해제 후 삭제해주세요.

<br>

## 아바타 삭제하기
``등록된 아바타 목록`` 에서 삭제할 아바타를 선택한 후, 탭 맨 아래의 ``삭제하기`` 버튼을 클릭합니다.

>[!WARNING]
> - 기본 아바타로 설정했거나, 삭제할 아바타가 유일한 (마지막) 아바타일 경우 삭제가 불가능합니다.
